Задана целевая функция:

f(x) = 2x1 + 2x2 => max

И система ограничений:

2x1 + 2x1 = 2
2x1 + 2x2 = 2

Решим данную задачу на максимум методом симплексных таблиц.
Построим первый опорный план:

x3	2	2	2	1	0	1

x4	2	2	2	0	1	1

f(x)	0	-2	-2	0	0	

Первый опорный план не является оптимальным, поскольку
в индексной строке присутствуют неположительные элементы.
Erо необходимо улучшить.

x1	1	1	1	0.5	0	

x4	0	0	0	-1	1	

f(x)	2	0	0	1	0	

Данный план является оптимальным и не требует улучшения. Решение найдено.
